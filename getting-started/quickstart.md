# 시작하기

SafeIntelligence를 처음 사용하시나요? 이 가이드를 따라 빠르게 시작할 수 있습니다.

## 1단계: 로그인

관리자로부터 받은 이메일과 비밀번호로 로그인합니다. 초대 링크를 받았다면 링크를 클릭하여 계정을 활성화하세요.

## 2단계: 요약 보고 확인

로그인 후 **요약 보고** 화면이 표시됩니다. 여기서 조직의 전체 보안 현황을 한눈에 파악할 수 있습니다.

* **즉시 조치 (Immediate Actions)**: 우선 처리가 필요한 TOP 5 발견사항
* **자산 노출 현황 (Asset Exposure)**: 도메인별 보안 등급과 발견사항 수
* **위협 인텔리전스 (Threat Intelligence)**: 다크웹 유출 현황과 AI 분석 결과

## 3단계: 도메인 스코프 설정

화면 상단의 **Active Scope** 카드를 클릭하여 특정 도메인을 선택하면, 모든 페이지의 데이터가 해당 도메인으로 필터링됩니다.

* 전체 조직 보기: 스코프를 해제하면 모든 도메인의 데이터가 표시됩니다.
* 특정 도메인 보기: 도메인을 선택하면 해당 도메인과 하위 서브도메인의 데이터만 표시됩니다.

## 4단계: 발견사항 확인 및 조치

1. **요약 보고** → **전체 파인딩 보기** 클릭
2. 심각도별 필터 적용 (Critical, High, Medium, Low)
3. 발견사항 클릭 → 상세 정보 확인
4. **상태 변경** 섹션에서 조치 상태 업데이트 (미조치 → 조치 중 → 조치 완료)

## 5단계: 다크웹 모니터링 확인

요약 보고의 **위협 인텔리전스** 섹션에서 **다크웹 모니터링 보기 >** 를 클릭하여 다크웹 유출 현황을 확인합니다.
