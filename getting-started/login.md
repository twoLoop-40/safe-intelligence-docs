# 로그인 및 계정

## 로그인

SafeIntelligence에 접속하면 로그인 화면이 나타납니다.

<figure><img src="../.gitbook/assets/스크린샷 2026-03-12 오전 8.07.37.png" alt=""><figcaption></figcaption></figure>

1. **이메일**: 등록된 이메일 주소 입력
2. **비밀번호**: 비밀번호 입력
3. **Sign In** 버튼 클릭

## 초대를 통한 가입

관리자가 보낸 초대 이메일의 링크를 클릭하면 계정 활성화 페이지로 이동합니다.

1. 초대 이메일에서 **계정 활성화** 링크 클릭
2. 비밀번호 설정
3. 로그인

## 세션 만료

보안을 위해 일정 시간 동안 활동이 없으면 세션이 만료됩니다. 자동으로 로그인 화면으로 이동하며, 다시 로그인하면 이전 페이지로 돌아갑니다.

## 비밀번호 변경

현재 비밀번호 변경은 관리자에게 요청하세요. (향후 셀프 서비스 기능 추가 예정)
