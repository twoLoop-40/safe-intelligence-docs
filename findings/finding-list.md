# 파인딩 목록

## 개요

조직의 모든 보안 발견사항을 통합 관리하는 페이지입니다. 사이드바의 **Findings** 메뉴에서 접근합니다.

## 테이블 구성

| 열 | 설명 |
|----|------|
| 제목 | 발견사항 이름 (클릭 → 상세 이동) |
| 심각도 | Critical / High / Medium / Low 배지 |
| 상태 | 미조치 / 조치 중 / 조치 완료 / 무시 / 오탐 |
| 도메인 | 관련 도메인 |
| 유형 | CVE, 포트, 인증서, 다크웹 등 |
| 발견일 | 처음 탐지된 날짜 |

## 필터

화면 상단의 필터 바에서 다양한 조건으로 필터링할 수 있습니다:

* **심각도**: Critical, High, Medium, Low 선택
* **상태**: 미조치, 조치 중, 조치 완료, 무시, 오탐
* **유형**: CVE, 포트 스캔, 인증서, 다크웹 등
* **도메인**: 특정 도메인 선택
* **검색**: 키워드로 제목 검색

{% hint style="info" %}
필터 설정은 URL에 저장됩니다. 필터가 적용된 URL을 공유하면 동일한 필터 상태로 볼 수 있습니다.
{% endhint %}

## 뷰 모드

* **테이블 뷰**: 기본 목록 형태
* **심각도별 뷰**: 심각도 기준 그룹핑

## 발견사항 클릭

테이블의 행을 클릭하면 **파인딩 상세** 페이지로 이동합니다.
