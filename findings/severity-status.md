# 심각도 및 상태 관리

## 심각도 (Severity)

발견사항의 위험도를 나타내는 4단계 분류입니다.

| 심각도 | 색상 | CVSS 범위 | 설명 |
|--------|------|-----------|------|
| Critical | 빨강 | 9.0-10.0 | 즉시 조치 필요. 원격 코드 실행, 인증 우회 등 |
| High | 주황 | 7.0-8.9 | 높은 위험. 정보 유출, 권한 상승 가능성 |
| Medium | 노랑 | 4.0-6.9 | 중간 위험. 제한적 영향 |
| Low | 파랑 | 0.1-3.9 | 낮은 위험. 정보 제공 수준 |

## 상태 (Status)

발견사항의 처리 상태를 추적합니다.

| 상태 | 배지 색상 | 설명 |
|------|----------|------|
| 미조치 (Open) | 빨강 | 새로 발견되어 아직 대응하지 않은 상태 |
| 조치 중 (In Progress) | 노랑 | 담당자가 조치를 진행 중인 상태 |
| 조치 완료 (Resolved) | 초록 | 문제가 해결된 상태 |
| 무시 (Ignored) | 회색 | 위험을 수용하고 의도적으로 무시 |
| 오탐 (False Positive) | 회색 | 실제 위협이 아닌 것으로 판단 |

## 점수 계산

### 보안 점수 (Security Score)

각 도메인의 보안 점수는 0~100점으로 계산됩니다:

* **높을수록 좋음**: 100점이 가장 안전
* **감점 방식**: 발견사항의 심각도에 따라 감점
* **시간 감소**: 오래된 발견사항은 감점 효과가 줄어듦 (temporal decay)

### 보안 등급

| 등급 | 점수 | 색상 |
|------|------|------|
| A | 75점 이상 | 초록 |
| B | 60-74점 | 연두 |
| C | 40-59점 | 노랑 |
| D | 20-39점 | 주황 |
| F | 19점 이하 | 빨강 |
