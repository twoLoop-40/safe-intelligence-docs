# 필터 및 검색

## 파인딩 필터

파인딩 목록 화면 상단의 필터 바를 사용하여 원하는 조건으로 필터링할 수 있습니다.

### 사용 가능한 필터

| 필터 | 옵션 | 설명 |
|------|------|------|
| 심각도 | Critical, High, Medium, Low | 하나 이상 선택 가능 |
| 상태 | Open, In Progress, Resolved, Ignored, False Positive | 하나 이상 선택 가능 |
| 유형 | CVE, Open Port, Expired Cert, Credential Leak 등 | 발견 유형별 필터 |
| 도메인 | 등록된 도메인 목록 | 특정 도메인만 보기 |

### 필터 조합

여러 필터를 동시에 적용할 수 있습니다. 예를 들어:
* Critical + Open = 아직 조치하지 않은 심각한 발견사항만 보기
* Dark Web + High = 높은 위험도의 다크웹 유출만 보기

### URL 필터 저장

필터를 적용하면 브라우저 URL에 자동으로 반영됩니다:
* 북마크하여 자주 사용하는 필터 조합을 저장
* URL을 공유하여 팀원과 동일한 뷰 공유

## 전체 검색 (Cmd+K)

화면 어디서든 **Cmd+K** (Mac) 또는 **Ctrl+K** (Windows)를 눌러 전체 검색을 실행할 수 있습니다.

검색 대상:
* 도메인 이름
* IP 주소
* CVE ID
* 이메일 주소
* 발견사항 제목

검색 결과를 클릭하면 해당 항목의 상세 페이지로 바로 이동합니다.
