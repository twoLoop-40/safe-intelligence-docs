# Filter & Search

## Finding Filters

Use the filter bar at the top of the Finding List screen to filter by your desired criteria.

### Available Filters

| Filter | Options | Description |
|------|------|------|
| Severity | Critical, High, Medium, Low | Multiple selections allowed |
| Status | Open, In Progress, Resolved, Ignored, False Positive | Multiple selections allowed |
| Type | CVE, Open Port, Expired Cert, Credential Leak, etc. | Filter by finding type |
| Domain | List of registered domains | View a specific domain only |

### Combining Filters

Multiple filters can be applied simultaneously. For example:
* Critical + Open = View only critical findings that have not been addressed
* Dark Web + High = View only high-risk dark web exposures

### URL Filter Persistence

Applied filters are automatically reflected in the browser URL:
* Bookmark to save frequently used filter combinations
* Share the URL to give team members the same filtered view

## Universal Search (Cmd+K)

Press **Cmd+K** (Mac) or **Ctrl+K** (Windows) from any screen to open universal search.

Search targets:
* Domain names
* IP addresses
* CVE IDs
* Email addresses
* Finding titles

Click a search result to navigate directly to that item's detail page.
