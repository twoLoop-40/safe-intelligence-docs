# Finding List

## Summary

A centralized page for managing all security findings across your organization. Access it from the **Findings** menu in the sidebar.

![Finding List](../assets/findings.png)

## Table Layout

| Column | Description |
|----|------|
| Title | Finding name (click to navigate to detail) |
| Severity | Critical / High / Medium / Low badge |
| Status | Open / In Progress / Resolved / Ignored / False Positive |
| Domain | Associated domain |
| Type | CVE, Port, Certificate, Dark Web, etc. |
| Discovered | Date first detected |

## Filters

Use the filter bar at the top of the screen to filter by various criteria:

* **Severity**: Select Critical, High, Medium, Low
* **Status**: Open, In Progress, Resolved, Ignored, False Positive
* **Type**: CVE, Open Port, Certificate, Dark Web, etc.
* **Domain**: Select a specific domain
* **Search**: Search titles by keyword

{% hint style="info" %}
Filter settings are saved in the URL. Share a filtered URL to allow team members to see the same filtered view.
{% endhint %}

## View Modes

* **Table View**: Default list format
* **Severity View**: Grouped by severity

## Clicking a Finding

Click a row in the table to navigate to the **Finding Detail** page.
