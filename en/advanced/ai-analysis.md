# AI Analysis

## Summary

SafeIntelligence leverages AI to provide in-depth analysis of security findings. Beyond simply listing vulnerabilities, it derives actionable insights about your organization's security posture.

## AI Insights (Overview)

View AI-generated security insights in the **Threat Intelligence** section of the Overview page:

* **Executive Summary**: A concise summary of the current security posture
* **Key Threats**: The top 3 most urgent security risks
* **Recommended Actions**: Immediately actionable response measures

### Generating Insights

Click the **Generate** button to produce AI insights based on the latest data. Generation takes approximately 10-30 seconds.

## AI Provider Settings

{% hint style="info" %}
AI provider settings require **Super Admin** privileges.
{% endhint %}

You can select the provider for AI analysis in the administrator settings:

* **OpenAI**: Uses GPT models
* **Anthropic**: Uses Claude models

After configuring the API key and model, AI analysis features become available.

## Usage Notes

* AI analysis results are for reference purposes; review by a security professional is recommended
* Analysis is limited to your organization's security data
* The scope of data transmitted externally depends on the AI provider configuration
