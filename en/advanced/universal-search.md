# Universal Search (Cmd+K)

## Summary

Press **Cmd+K** (Mac) or **Ctrl+K** (Windows) to open universal search. Quickly search across all assets including domains, subdomains, findings, and IP addresses.

![Universal Search](../assets/universal-search.png)

## How to Use

1. Open the search modal with the **Cmd+K** shortcut
2. Enter your search query
3. Results are displayed in real time
4. Click a result to navigate to its detail page

## Search Targets

| Category | Search Target | Example |
|----------|----------|------|
| Domain | Domain name | `example.com` |
| Subdomain | Subdomain name | `api.example.com` |
| Finding | Title, CVE ID | `CVE-2024-1234` |
| IP Address | IPv4/IPv6 | `203.0.113.1` |

## Search Results

Results are displayed grouped by category:

* **Domains**: Domain name + score
* **Subdomains**: Subdomain name + parent domain
* **Findings**: Title + severity badge
* **IPs**: IP address + associated subdomain

## Keyboard Navigation

* **Up/Down Arrows**: Browse through results
* **Enter**: Navigate to the selected item
* **Esc**: Close the search modal
