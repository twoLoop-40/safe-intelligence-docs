# Scan History

## Summary

View the history of all past scans. Scan records are retained for up to 6 months.

## History Information

Each scan record includes the following information:

| Field | Description |
|------|------|
| Start Time | Date and time the scan started |
| End Time | Date and time the scan finished |
| Target Domain | The domain that was scanned |
| Status | Completed / Failed / Canceled |
| Items Found | Number of newly discovered security items |

## Automatic Scans

SafeIntelligence performs automatic scans on registered domains:

* **Security Scans**: Periodic automated execution
* **Dark Web Monitoring**: Automatic refresh every 6 hours
* **Certificate Expiration Check**: Checked daily

## Retention Period

* Scan results: Retained for **6 months**
* Findings: Retained permanently until resolved
* Score history: Retained permanently
