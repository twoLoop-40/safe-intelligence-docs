# Glossary

## General Terms

| Term | Definition |
|------|------|
| **ASM** | Attack Surface Management. The practice of identifying externally exposed assets and managing security risks |
| **OSINT** | Open Source Intelligence. Intelligence gathering using publicly available information |
| **CTI** | Cyber Threat Intelligence. Intelligence related to cyber threats |

## Asset-Related

| Term | Definition |
|------|------|
| **Domain** | An internet domain owned by the organization (e.g., example.com) |
| **Subdomain** | A hostname under a domain (e.g., api.example.com) |
| **Scope** | The range of domains configured for monitoring |
| **Asset** | Resources under monitoring, such as domains, subdomains, and IPs |

## Vulnerability-Related

| Term | Definition |
|------|------|
| **CVE** | Common Vulnerabilities and Exposures. Identifiers for publicly known security vulnerabilities |
| **CVSS** | Common Vulnerability Scoring System. Vulnerability severity score (0-10) |
| **CWE** | Common Weakness Enumeration. A classification system for software security weaknesses |
| **CPE** | Common Platform Enumeration. A standard for identifying software and hardware |

## Severity

| Grade | CVSS Range | Description |
|------|----------|------|
| **Critical** | 9.0 - 10.0 | Critical vulnerability requiring immediate response |
| **High** | 7.0 - 8.9 | High risk requiring prompt response |
| **Medium** | 4.0 - 6.9 | Moderate risk requiring planned response |
| **Low** | 0.1 - 3.9 | Low risk, monitoring level |

## Dark Web-Related

| Term | Definition |
|------|------|
| **Dark Web** | An area of the internet not accessible through standard browsers |
| **Credential Leak** | A case where email + password is found on the dark web |
| **Credential Stuffing** | An attack that uses leaked credentials to attempt login on other services |
| **Compromised Device** | A device infected with malware that has had information leaked |
| **Phishing Kit** | A collection of tools used for phishing attacks |

## Security Grade

| Grade | Score Range | Meaning |
|------|----------|------|
| **A** | 75 or above | Excellent security posture |
| **B** | 60 - 74 | Good security posture |
| **C** | 40 - 59 | Security posture needs improvement |
| **D** | 20 - 39 | Vulnerable security posture |
| **F** | Below 20 | Critically vulnerable security posture |
