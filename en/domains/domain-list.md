# Domain List

## Summary

Manage all monitored domains from a single screen. Access it from the **Domains** menu in the sidebar.

![Domain List](../assets/domains.png)

## Domain Card

Each domain is displayed as a card:

* **Security Grade** (A-F): Color-coded badge
* **Domain Name**: Monospace font
* **Subdomain Count**: Number of child subdomains
* **Critical/High Findings**: Severity badges
* **Total Finding Count**: Large number
* **Last Scan Time**: Relative time

## View Domain Detail

Click a card to navigate to the **Domain Detail** page.

## Add Domain

Click the **+ Add Domain** button to register a new domain. For more information, see the [Add Domain](add-domain.md) page.

## Sorting and Filtering

* Sort by security grade, finding count, domain name, and more
* Use the search bar to quickly find a specific domain
