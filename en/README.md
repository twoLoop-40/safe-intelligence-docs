# SafeIntelligence

**Enterprise Security Intelligence Platform**

SafeIntelligence is an OSINT (Open Source Intelligence)-based B2B enterprise security assessment platform. It comprehensively analyzes your organization's security posture centered around domains and monitors threats in real time.

## Key Features

### Overview
A comprehensive view of your organization's security posture at a glance. Provides immediate action items, asset exposure status, and dark web threat intelligence across three sections.

### Domain Management (Domains)
Manage the domains and subdomains under monitoring. View each domain's security grade, number of findings, and most recent scan time.

### Security Findings (Findings)
Centrally manage all security issues including CVE vulnerabilities, certificate expirations, and dark web exposures. Supports severity-based filtering and status management.

### Dark Web Monitoring (Dark Web)
Track dark web threats in real time through ZeroDarkWeb integration, including credential leaks, URL exposures, and compromised devices.

### Scan Management (Scanning)
Execute security scans on domains and subdomains and review results. Supports Nmap port scanning, SSL verification, CVE detection, and more.

### Universal Search (Cmd+K)
Search across all entities (domains, IPs, CVEs, emails, etc.) and explore associated assets.
