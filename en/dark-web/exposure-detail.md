# Exposure Card Detail

## Opening the Drawer

Click an exposure card on the Dark Web Gateway to open a detail drawer that slides in from the right.

## Category-Specific Details

### Credential Leaks (Credentials)

* **Password Type Analysis**: Plaintext vs. Hash ratio
* **Subdomain Breakdown**: Which subdomains had credentials leaked
* **Event Timeline**: Chronological list of leak events
* **Leak Sources**: Which dark web forums/markets the credentials were found on

### Compromised Devices (Devices)

* **Event List**: Compromised device information
* **Timeline**: When the infection was discovered

### URL Leaks

* **Subdomain Breakdown**: List of subdomains with URL exposures
* **Event List**: Leaked URL information

### Document/Email Leaks (Documents/Email)

* **Event List**: Leaked document and email information
* **Accordion Details**: Expanded details for each event

## Closing the Drawer

* Click the X button
* Click outside the drawer area
* Press the ESC key
