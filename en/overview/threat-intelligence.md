# Threat Intelligence

## Summary

The third section of the Overview displays dark web threat information and AI analysis results.

## 4 KPI Metrics

| Metric | Description | Color |
|--------|------------|-------|
| CREDENTIALS LEAKED | Number of leaked credentials (email + password) | Red |
| DARK WEB MENTIONS | Number of times the organization is mentioned on the dark web | Cyan |
| COMPROMISED ACCOUNTS | Number of unique email accounts at risk | Yellow |
| THREAT SOURCES DETECTED | Number of detected threat sources (forums, markets, etc.) | Green |

Each metric is highlighted in its respective color when the value is greater than zero.

## View Dark Web Monitoring

Click **View Dark Web Monitoring >** to navigate to the dedicated dark web monitoring page. On this page, you can review detailed exposure information and recommended response actions.

## AI Insights

AI analyzes the current security situation and generates an executive-level summary.

* **Summary**: Description of the current threat level and key issues
* **Recommended Actions**: Action items with severity-colored borders (up to 3)
  * Red border: Immediate response required
  * Orange border: High priority
  * Yellow border: Medium priority

Click the **Generate AI Report** button to create a detailed intelligence report.

## Detected Sources

A threat source list displayed in the right sidebar. Each source includes:

* **Source Name**: e.g., BreachForum, Telegram #leaks
* **Severity Badge**: CRITICAL / HIGH
* **Category Badge**: e.g., Credential Dump, Corporate Mention
