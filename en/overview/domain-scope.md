# Domain Scope Settings

## What is Active Scope?

**Active Scope** determines the range of data displayed on the current screen. When a specific domain is selected, all pages display only data related to that domain.

## Usage

### Selecting a Domain

1. Click the **ACTIVE SCOPE** card in the top header area
2. Select the desired domain from the dropdown
3. Data across all pages (Overview, Findings, Dark Web, etc.) is filtered immediately

### Clearing the Scope

* **View All**: Click "All" or the X button on the scope card
* When the scope is cleared, data for the entire organization is displayed

## Affected Pages

Setting the Active Scope affects the following pages:

| Page | Effect |
|------|--------|
| Overview | TOP 5, statistics, and all KPIs filtered to the selected domain |
| Finding List | Only findings for the selected domain are displayed |
| Dark Web | Only dark web exposures for the selected domain are displayed |
| Penetration Testing | The selected domain is automatically chosen |

## Tips

* Use this feature when you want to focus on the security status of a specific domain
* When preparing reports, switch the scope between domains and capture screenshots for an efficient workflow
