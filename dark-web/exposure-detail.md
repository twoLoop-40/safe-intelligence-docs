# 노출 카드 상세

## 드로어 열기

다크웹 게이트웨이에서 노출 카드를 클릭하면 오른쪽에서 상세 드로어가 슬라이드 인합니다.

## 카테고리별 상세 정보

### 자격증명 유출 (Credentials)

* **비밀번호 유형 분석**: 평문(Plaintext) vs 해시(Hash) 비율
* **서브도메인별 분류**: 어떤 서브도메인의 자격증명이 유출되었는지
* **이벤트 타임라인**: 유출 이벤트의 시간순 목록
* **유출 소스**: 어떤 다크웹 포럼/마켓에서 발견되었는지

### 감염 디바이스 (Devices)

* **이벤트 목록**: 감염된 디바이스 정보
* **타임라인**: 감염 발견 시점

### URL 유출 (URL Leaks)

* **서브도메인별 분류**: URL이 유출된 서브도메인 목록
* **이벤트 목록**: 유출된 URL 정보

### 문서/이메일 유출 (Documents/Email)

* **이벤트 목록**: 유출된 문서 및 이메일 정보
* **아코디언 상세**: 각 이벤트의 세부 정보

## 드로어 닫기

* X 버튼 클릭
* 드로어 바깥 영역 클릭
* ESC 키
