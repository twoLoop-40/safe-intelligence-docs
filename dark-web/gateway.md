# 다크웹 게이트웨이

## 개요

다크웹 모니터링 페이지는 ZeroDarkWeb(ZDW) 연동을 통해 조직의 다크웹 노출 현황을 보여줍니다. 사이드바의 **Dark Web** 메뉴에서 접근하거나, 요약 보고의 **다크웹 모니터링 보기 >** 링크를 클릭합니다.

## 화면 구성

### 상단: 노출 통계

4개의 카테고리별 노출 수를 보여줍니다:

| 카테고리 | 설명 |
|----------|------|
| Credentials | 유출된 자격증명 (이메일+비밀번호) |
| Devices | 감염된 디바이스 |
| URL Leaks | 유출된 URL 정보 |
| Documents/Email | 유출된 문서 및 이메일 |

### 노출 카드 목록

각 노출 이벤트가 카드 형태로 표시됩니다:

* **유형 아이콘**: 자격증명, 디바이스, URL, 문서/이메일
* **이메일/도메인**: 관련 이메일 주소 또는 도메인
* **유출 소스**: 유출이 발견된 출처
* **날짜**: 유출 날짜
* **심각도**: 위험도 표시

### 노출 카드 클릭

카드를 클릭하면 오른쪽에서 **상세 드로어**가 열립니다. 드로어에는 카테고리별로 다른 정보가 표시됩니다:

* **자격증명**: 비밀번호 유형(평문/해시), 유출 소스 분석, 서브도메인별 분류
* **디바이스**: 감염 이벤트 목록
* **URL 유출**: 서브도메인별 분류, 이벤트 목록
* **문서/이메일**: 이벤트 목록, 상세 정보 아코디언

## 데이터 갱신

다크웹 데이터는 **6시간마다** 자동으로 갱신됩니다. 하단의 안내 메시지에서 갱신 주기를 확인할 수 있습니다.

## CTA 섹션

페이지 하단에 전체 ZeroDarkWeb 플랫폼으로의 링크가 제공됩니다. 현재 페이지는 최근 이벤트의 스냅샷이며, 전체 이력과 고급 분석은 ZDW 플랫폼에서 확인할 수 있습니다.
