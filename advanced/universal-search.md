# 검색 (Cmd+K)

## 개요

**Cmd+K** (Mac) 또는 **Ctrl+K** (Windows)를 눌러 통합 검색을 열 수 있습니다. 도메인, 서브도메인, 발견사항, IP 주소 등 모든 자산을 빠르게 검색합니다.

## 사용 방법

1. **Cmd+K** 단축키로 검색 모달을 엽니다
2. 검색어를 입력합니다
3. 실시간으로 결과가 표시됩니다
4. 결과를 클릭하면 해당 상세 페이지로 이동합니다

## 검색 대상

| 카테고리  | 검색 대상      | 예시                |
| ----- | ---------- | ----------------- |
| 도메인   | 도메인 이름     | `example.com`     |
| 서브도메인 | 서브도메인 이름   | `api.example.com` |
| 발견사항  | 제목, CVE ID | `CVE-2024-1234`   |
| IP 주소 | IPv4/IPv6  | `203.0.113.1`     |

## 검색 결과

결과는 카테고리별로 그룹화되어 표시됩니다:

* **도메인**: 도메인 이름 + 스코어
* **서브도메인**: 서브도메인 이름 + 상위 도메인
* **발견사항**: 제목 + 심각도 배지
* **IP**: IP 주소 + 관련 서브도메인

## 키보드 네비게이션

* **↑↓**: 결과 목록 탐색
* **Enter**: 선택한 항목으로 이동
* **Esc**: 검색 모달 닫기
