# AI 분석

## 개요

SafeIntelligence는 AI를 활용하여 보안 발견사항에 대한 심층 분석을 제공합니다. 단순한 취약점 나열을 넘어, 조직의 보안 상태에 대한 인사이트를 도출합니다.

## AI 인사이트 (요약 보고)

요약 보고(Overview) 페이지의 **위협 인텔리전스** 섹션에서 AI가 생성한 보안 인사이트를 확인할 수 있습니다:

* **경영진 요약**: 현재 보안 상태에 대한 간결한 요약
* **주요 위협**: 가장 시급한 보안 위험 3가지
* **권장 조치**: 즉시 실행 가능한 대응 방안

### 인사이트 생성

**Generate** 버튼을 클릭하면 최신 데이터를 기반으로 AI 인사이트가 생성됩니다. 생성에는 약 10-30초가 소요됩니다.

## AI 제공자 설정

{% hint style="info" %}
AI 제공자 설정은 **슈퍼 관리자** 권한이 필요합니다.
{% endhint %}

관리자 설정에서 AI 분석에 사용할 제공자를 선택할 수 있습니다:

* **OpenAI**: GPT 모델 사용
* **Anthropic**: Claude 모델 사용

API 키와 모델을 설정한 후 AI 분석 기능을 사용할 수 있습니다.

## 사용 시 주의사항

* AI 분석 결과는 참고 자료이며, 전문가의 검토가 권장됩니다
* 분석에 사용되는 데이터는 조직의 보안 데이터에 한정됩니다
* 외부로 데이터가 전송되는 범위는 AI 제공자 설정에 따릅니다
