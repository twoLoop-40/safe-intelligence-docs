# 検出事項一覧

## 概要

組織のすべてのセキュリティ検出事項を統合管理するページです。サイドバーの**Findings**メニューからアクセスします。

![検出事項一覧画面](../assets/findings.png)

## テーブル構成

| 列 | 説明 |
|----|------|
| タイトル | 検出事項名（クリック → 詳細に遷移） |
| 深刻度 | Critical / High / Medium / Low バッジ |
| ステータス | 未対処 / 対処中 / 対処完了 / 無視 / 誤検知 |
| ドメイン | 関連ドメイン |
| 種別 | CVE、ポート、証明書、ダークウェブなど |
| 検出日 | 初めて検出された日付 |

## フィルター

画面上部のフィルターバーで、さまざまな条件でフィルタリングできます：

* **深刻度**: Critical、High、Medium、Low を選択
* **ステータス**: 未対処、対処中、対処完了、無視、誤検知
* **種別**: CVE、ポートスキャン、証明書、ダークウェブなど
* **ドメイン**: 特定のドメインを選択
* **検索**: キーワードでタイトルを検索

{% hint style="info" %}
フィルター設定はURLに保存されます。フィルターが適用されたURLを共有すると、同じフィルター状態で表示できます。
{% endhint %}

## 表示モード

* **テーブルビュー**: 標準的な一覧形式
* **深刻度別ビュー**: 深刻度でグループ化された表示

## 検出事項のクリック

テーブルの行をクリックすると、**検出事項詳細**ページに遷移します。
