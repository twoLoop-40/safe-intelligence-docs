# フィルターと検索

## 検出事項フィルター

検出事項一覧画面上部のフィルターバーを使用して、希望する条件でフィルタリングできます。

### 利用可能なフィルター

| フィルター | オプション | 説明 |
|------|------|------|
| 深刻度 | Critical, High, Medium, Low | 複数選択可能 |
| ステータス | Open, In Progress, Resolved, Ignored, False Positive | 複数選択可能 |
| 種別 | CVE, Open Port, Expired Cert, Credential Leak など | 検出種別でフィルター |
| ドメイン | 登録済みドメインの一覧 | 特定ドメインのみ表示 |

### フィルターの組み合わせ

複数のフィルターを同時に適用できます。例：
* Critical + Open = まだ対処していない深刻な検出事項のみ表示
* Dark Web + High = 高リスクのダークウェブ流出のみ表示

### URLフィルターの保存

フィルターを適用すると、ブラウザのURLに自動的に反映されます：
* ブックマークして、よく使うフィルターの組み合わせを保存
* URLを共有して、チームメンバーと同じビューを共有

## 統合検索 (Cmd+K)

画面のどこからでも**Cmd+K**（Mac）または**Ctrl+K**（Windows）を押して統合検索を実行できます。

検索対象：
* ドメイン名
* IPアドレス
* CVE ID
* メールアドレス
* 検出事項のタイトル

検索結果をクリックすると、該当項目の詳細ページに直接遷移します。
