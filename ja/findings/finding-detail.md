# 検出事項詳細

## 概要

個々のセキュリティ検出事項の詳細情報を確認し、対処ステータスを管理するページです。

## 画面構成

### 上部：パンくずリスト

`← 戻る > ドメイン > 検出事項タイトル` 形式のパスが表示されます。ドメインをクリックすると、該当ドメインの詳細に遷移します。

### ヘッダーカード

* **タイトル**: 検出事項名
* **深刻度バッジ**: Critical（赤）、High（オレンジ）、Medium（黄）、Low（青）
* **ステータスバッジ**: 未対処、対処中、対処完了 など
* **種別バッジ**: CVE、認証情報流出、ポートスキャン など
* **CVE ID**: NVDリンク（該当する場合）
* **検出日**: 初めて検出された日付と時刻

### メインコンテンツ（左側）

| セクション | 内容 |
|------|------|
| 説明 | 検出事項の詳細な説明 |
| 改善方法 | 推奨される対処方法 |
| 証拠 | 技術的な証拠（ソース、日付など） |
| CPE識別子 | 該当するソフトウェア識別子 (CPE) |
| 参考情報 | NVD、CVE.org などの外部参照リンク |

### サイドバー（右側）

| セクション | 内容 |
|------|------|
| CVSSスコア | CVSSスコアバー + 深刻度ラベル |
| CVSSベクター | 攻撃ベクター、複雑度など8項目 |
| CWE分類 | CWE ID + 名称 + Top 25該当有無 |
| 悪用確率 | EPSS推定確率（CVEのみ該当） |
| 影響を受けるコンポーネント | 影響を受けるシステムコンポーネント |
| メタデータ | ID、種別、資産情報 |
| ステータス変更 | ステータス更新ボタン |
| AI分析 | AIベースの分析結果（提供時） |

## ステータスの変更

右側サイドバーの**ステータス変更**セクションで、検出事項の対処状況を変更できます：

| ステータス | 説明 |
|------|------|
| 未対処 (Open) | まだ対応していない状態 |
| 対処中 (In Progress) | 現在対応を進めている状態 |
| 対処完了 (Resolved) | 問題が解決された状態 |
| 無視 (Ignored) | 意図的に無視（リスク受容） |
| 誤検知 (False Positive) | 実際の脅威ではないと判断 |

{% hint style="warning" %}
対処完了、無視、誤検知に変更する際は、理由を入力できます。監査証跡のために理由を記録することを推奨します。
{% endhint %}
