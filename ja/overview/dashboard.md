# サマリーレポート画面

**サマリーレポート**（Overview）は、ログイン後に最初に表示されるメイン画面です。組織のセキュリティ状況を3つのセクションで要約します。

![サマリーレポートダッシュボード](../assets/overview.png)

## 画面構成

### 上部：Active Scope

画面上部に、現在選択されているドメインスコープが表示されます。クリックして特定のドメインを選択すると、すべてのデータがそのドメインでフィルタリングされます。

### セクション01：即時対応 (Immediate Actions)

最も緊急性の高いセキュリティ問題TOP 5をテーブル形式で表示します。

| 列 | 説明 |
|---|---|
| 検出事項タイトル | 問題名（クリックすると詳細ページに遷移） |
| 深刻度 | Critical / High / Medium / Low |
| ドメイン | 影響を受けるドメイン |
| リスクスコア | 深刻度 × 資産重要度に基づくスコア |

**すべての検出事項を表示** リンクをクリックすると、検出事項一覧ページに遷移します。

下部には**ドメインリスクマップ**と**サブドメインリスクプレビュー**が表示されます。

### セクション02：資産エクスポージャー (Asset Exposure)

監視中のドメインをカードグリッドで表示します。各カードには以下が含まれます：

* **セキュリティ等級** (A～F) — 色分けされた円形バッジ
* **ドメイン名**
* **サブドメイン数**
* **Critical / High 検出事項数** — 深刻度バッジ
* **検出事項の総数**
* **最終スキャン日時**

カードをクリックすると、該当ドメインの詳細ダッシュボードに遷移します。

### セクション03：脅威インテリジェンス (Threat Intelligence)

ダークウェブの脅威情報とAI分析結果を表示します。

**4つのKPI指標：**
* **CREDENTIALS LEAKED**: 流出した認証情報の数（赤色）
* **DARK WEB MENTIONS**: ダークウェブでの言及数（シアン色）
* **COMPROMISED ACCOUNTS**: 危険にさらされたアカウント数（黄色）
* **THREAT SOURCES DETECTED**: 検出された脅威ソースの数（緑色）

**ダークウェブモニタリングを表示 >** リンクをクリックすると、ダークウェブモニタリングページに遷移します。

下部には**AIインサイト**（AI分析サマリー）と**検出されたソース**（脅威ソース一覧）サイドバーが表示されます。

## データの更新

サマリーレポートのデータはリアルタイムで更新されます。ページを再読み込みするか、ドメインスコープを変更すると、最新のデータを確認できます。
