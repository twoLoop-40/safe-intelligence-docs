# ドメイン詳細

## 概要

特定のドメインのセキュリティ状況を詳しく確認するページです。ドメイン一覧またはサマリーレポートからドメインをクリックすると遷移します。

![ドメイン詳細画面](../assets/domain-detail.png)

## 画面構成

### 上部：ドメイン情報

* **ドメイン名**: 現在閲覧中のドメイン
* **セキュリティスコア**: 0～100の数値と等級 (A～F)
* **統計カード**: サブドメイン数、検出事項数、最新スキャン日時

### タブ構成

| タブ | 内容 |
|----|------|
| Overview | ドメインのセキュリティ概要、深刻度分布、最近のアクティビティ |
| Findings | このドメインのすべての検出事項一覧 |
| Dark Web | このドメインに関するダークウェブ流出状況 |
| Subdomains | 配下のサブドメイン一覧とステータス |

### Attack Surfaceセクション

ドメインの攻撃対象領域をビジュアルで表示します：

* **オープンポート**: 検出されたネットワークサービスの一覧
* **SSL証明書**: 証明書のステータスと有効期限
* **Webテクノロジー**: 検出されたWebサーバー、フレームワークなど

## サブドメインの探索

Subdomainsタブで個々のサブドメインをクリックすると、サブドメイン詳細ページに遷移します。各サブドメインの：

* IPアドレス
* サービスステータス
* セキュリティ検出事項
* ポート情報

を確認できます。

## スキャンの実行

ドメイン詳細ページの**スキャン開始**ボタンをクリックして、該当ドメインに対するセキュリティスキャンをすぐに実行できます。
