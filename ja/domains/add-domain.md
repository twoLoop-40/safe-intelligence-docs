# ドメインの追加

## 新規ドメインの登録

監視対象のドメインを追加する方法です。

### 方法1：ドメイン一覧から追加

1. サイドバーで**Domains**をクリック
2. **+ ドメインを追加** ボタンをクリック
3. ドメイン名を入力（例: example.com）
4. **追加** ボタンをクリック

### 方法2：スキャンによる自動追加

新規スキャンの実行時にドメインを入力すると、自動的に登録されます。

1. サイドバーで**New Scan**をクリック
2. 対象ドメインを入力
3. スキャンを実行 → ドメインとサブドメインが自動的に登録

## 注意事項

* 自組織が所有するドメインのみを登録してください
* 登録されたドメインはすべてのチームメンバーに共有されます
* ドメインごとのサブドメインは自動探索されます（Subfinder ベース）
