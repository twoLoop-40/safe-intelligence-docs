# サブドメイン管理

## 概要

各ドメイン配下のサブドメインを管理・監視します。ドメイン詳細ページの**Subdomains**タブで確認できます。

## サブドメイン一覧

各サブドメインについて、以下の情報が表示されます：

| 列 | 説明 |
|----|------|
| 名前 | サブドメインアドレス (例: api.example.com) |
| IP | 確認されたIPアドレス |
| ステータス | アクティブ/非アクティブ |
| Critical | Critical検出事項の数 |
| High | High検出事項の数 |
| ポート | オープンポートの数 |
| SSL | SSL証明書のステータス |
| 最新スキャン | 最終スキャン日時 |

## サブドメイン詳細

サブドメインをクリックすると詳細ページに遷移します：

* **基本情報**: IP、位置情報 (GeoIP)、ホスティング情報
* **ポート情報**: オープンポートと実行中のサービス
* **検出事項**: 該当サブドメインのセキュリティ問題
* **SSL証明書**: 証明書の詳細情報と有効期限

## サブドメインの追加

ドメイン詳細ページからサブドメインを手動で追加できます：

1. Subdomainsタブに移動
2. **+ サブドメインを追加** 入力フィールドにサブドメイン名を入力
3. 追加ボタンをクリック

{% hint style="info" %}
セキュリティスキャンを実行すると、サブドメインが自動的に検出・追加されます。
{% endhint %}
