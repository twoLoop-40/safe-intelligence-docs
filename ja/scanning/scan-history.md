# スキャン履歴

## 概要

過去に実行されたすべてのスキャンの履歴を確認できます。最大6か月分のスキャン記録が保管されます。

## 履歴情報

各スキャン記録には以下の情報が含まれます：

| 項目 | 説明 |
|------|------|
| 開始時刻 | スキャンが開始された日時 |
| 完了時刻 | スキャンが終了した日時 |
| 対象ドメイン | スキャン対象のドメイン |
| ステータス | 完了 / 失敗 / キャンセル |
| 検出件数 | 新たに検出されたセキュリティ事項の数 |

## 自動スキャン

SafeIntelligenceは、登録されたドメインに対して自動スキャンを実行します：

* **セキュリティスキャン**: 定期的に自動実行
* **ダークウェブモニタリング**: 6時間周期で自動更新
* **証明書有効期限の確認**: 毎日確認

## 保管期間

* スキャン結果: **6か月**保管
* 検出事項 (Findings): 解決されるまで永続保管
* スコア履歴: 永続保管
