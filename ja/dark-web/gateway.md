# ダークウェブゲートウェイ

## 概要

ダークウェブモニタリングページは、ZeroDarkWeb (ZDW) 連携を通じて組織のダークウェブエクスポージャー状況を表示します。サイドバーの**Dark Web**メニューからアクセスするか、サマリーレポートの**ダークウェブモニタリングを表示 >** リンクをクリックします。

![ダークウェブゲートウェイ画面](../assets/dark-web.png)

## 画面構成

### 上部：エクスポージャー統計

4つのカテゴリ別のエクスポージャー件数を表示します：

| カテゴリ | 説明 |
|----------|------|
| Credentials | 流出した認証情報（メールアドレス＋パスワード） |
| Devices | 感染したデバイス |
| URL Leaks | 流出したURL情報 |
| Documents/Email | 流出した文書およびメール |

### エクスポージャーカード一覧

各エクスポージャーイベントがカード形式で表示されます：

* **種別アイコン**: 認証情報、デバイス、URL、文書/メール
* **メールアドレス/ドメイン**: 関連するメールアドレスまたはドメイン
* **流出元**: 流出が発見されたソース
* **日付**: 流出日
* **深刻度**: リスクレベル表示

### エクスポージャーカードのクリック

カードをクリックすると、右側から**詳細ドロワー**が表示されます。ドロワーにはカテゴリ別に異なる情報が表示されます：

* **認証情報**: パスワード種別（平文/ハッシュ）、流出元分析、サブドメイン別分類
* **デバイス**: 感染イベント一覧
* **URL流出**: サブドメイン別分類、イベント一覧
* **文書/メール**: イベント一覧、詳細情報アコーディオン

## データの更新

ダークウェブデータは**6時間ごと**に自動更新されます。下部のメッセージで更新周期を確認できます。
