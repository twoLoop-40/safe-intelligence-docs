# ワークスペース管理

## ワークスペースとは

ワークスペースは、ドメイン、スキャン、検出結果を管理するための独立した環境です。各ワークスペースには、独自のドメインセット、スキャン履歴、セキュリティデータがあります。

![ワークスペース切り替え](../../assets/workspace-switcher.png)

## ワークスペースの切り替え

1. サイドバーの**ワークスペース切り替え**（ロゴの下）をクリックします
2. 切り替え先のワークスペースを選択します
3. すべてのデータ（ドメイン、脆弱性、ダークウェブ）が選択したワークスペースの内容に更新されます

## ワークスペースの管理

ワークスペースドロップダウンの下部にある**ワークスペース管理**をクリックすると、以下の操作が可能です：

- 利用可能なすべてのワークスペースの表示
- 新しいワークスペースの作成
- ワークスペース設定の編集
- ワークスペースプランの確認（Basic、Professional、Enterprise）

## ワークスペースプラン

| 機能 | Basic | Professional | Enterprise |
|------|-------|-------------|------------|
| ドメイン数 | 最大5件 | 最大50件 | 無制限 |
| 月間スキャン回数 | 10回 | 100回 | 無制限 |
| ダークウェブ監視 | 制限あり | フル | フル + API |
| AIインサイト | --- | スタンダード | カスタムモデル |
| ユーザー数 | 3名 | 20名 | 無制限 |
