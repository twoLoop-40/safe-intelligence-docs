# AI分析

## 概要

SafeIntelligenceはAIを活用して、セキュリティ検出事項に対する詳細な分析を提供します。単なる脆弱性の一覧にとどまらず、組織のセキュリティ状態に関するインサイトを導出します。

## AIインサイト（サマリーレポート）

サマリーレポート (Overview) ページの**脅威インテリジェンス**セクションで、AIが生成したセキュリティインサイトを確認できます：

* **経営層向けサマリー**: 現在のセキュリティ状態に関する簡潔な要約
* **主要な脅威**: 最も緊急性の高いセキュリティリスク3件
* **推奨アクション**: 即座に実行可能な対応策

### インサイトの生成

**Generate** ボタンをクリックすると、最新のデータに基づいてAIインサイトが生成されます。生成には約10～30秒かかります。

## AIプロバイダーの設定

{% hint style="info" %}
AIプロバイダーの設定には、**スーパー管理者**権限が必要です。
{% endhint %}

管理者設定から、AI分析に使用するプロバイダーを選択できます：

* **OpenAI**: GPTモデルを使用
* **Anthropic**: Claudeモデルを使用

APIキーとモデルを設定した後、AI分析機能をご利用いただけます。

## ご利用上の注意

* AI分析結果は参考資料であり、専門家によるレビューを推奨します
* 分析に使用されるデータは、組織のセキュリティデータに限定されます
* 外部へのデータ送信範囲は、AIプロバイダーの設定に依存します
