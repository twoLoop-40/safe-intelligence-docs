# 検索 (Cmd+K)

## 概要

**Cmd+K**（Mac）または**Ctrl+K**（Windows）を押して統合検索を開くことができます。ドメイン、サブドメイン、検出事項、IPアドレスなど、すべての資産をすばやく検索します。

![統合検索画面](../assets/universal-search.png)

## 使い方

1. **Cmd+K** ショートカットで検索モーダルを開きます
2. 検索キーワードを入力します
3. リアルタイムで結果が表示されます
4. 結果をクリックすると、該当する詳細ページに遷移します

## 検索対象

| カテゴリ | 検索対象 | 例 |
|----------|----------|------|
| ドメイン | ドメイン名 | `example.com` |
| サブドメイン | サブドメイン名 | `api.example.com` |
| 検出事項 | タイトル、CVE ID | `CVE-2024-1234` |
| IPアドレス | IPv4/IPv6 | `203.0.113.1` |

## 検索結果

結果はカテゴリ別にグループ化されて表示されます：

* **ドメイン**: ドメイン名 + スコア
* **サブドメイン**: サブドメイン名 + 上位ドメイン
* **検出事項**: タイトル + 深刻度バッジ
* **IP**: IPアドレス + 関連するサブドメイン

## キーボードナビゲーション

* **↑↓**: 結果リストを移動
* **Enter**: 選択した項目に遷移
* **Esc**: 検索モーダルを閉じる
