# SafeIntelligence

**OSINTベースのエンタープライズセキュリティインテリジェンスプラットフォーム**

SafeIntelligenceは、組織の外部攻撃対象領域の監視・評価・管理を支援する包括的なセキュリティインテリジェンスソリューションです。オープンソースインテリジェンス（OSINT）技術を活用し、ドメイン、サブドメイン、ダークウェブ上の露出状況にわたって、組織のセキュリティ態勢を継続的に可視化します。

## 主な機能

### ドメイン中心のセキュリティ評価
組織に関連するすべてのドメインとサブドメインを自動的に検出・監視します。すべてのアセットに対して、リアルタイムのセキュリティグレード、脆弱性数、リスクスコアを提供します。

### ダークウェブインテリジェンス
組織のドメインに関連する漏洩クレデンシャル、公開URL、感染デバイス、流出ドキュメントについて、ダークウェブソースを監視します。

### 脆弱性管理
深刻度別の分類（Critical、High、Medium、Low）により、セキュリティ検出結果を追跡・優先順位付けします。CVEの詳細、CWEパターン、実行可能な修復ガイダンスを確認できます。

### ペネトレーションテスト
ドメインに対して自動ペネトレーションテストを実行し、概念実証（PoC）のエビデンスとリスク評価を含む詳細なレポートを確認できます。

### AI搭載インサイト
設定可能なAIプロバイダー（OpenAI、Anthropicなど）を活用して、経営層向けのセキュリティサマリーおよび脅威インテリジェンスレポートを生成します。

### ユニバーサル検索
キーボードショートカット1つ（Cmd+K / Ctrl+K）で、ドメイン、IPアドレス、CVE、メールアドレスなど、すべてのエンティティタイプを横断的に検索できます。エンティティピボットパネルを使用して、アセット間の関連性をたどることができます。

## アーキテクチャ

| レイヤー | テクノロジー |
|---------|------------|
| フロントエンド | React + TypeScript + Tailwind CSS |
| バックエンド | Python FastAPI + PostgreSQL |
| セキュリティデータ | OSINTスキャナー、ZDW API、CVE/NVD |
| AI | マルチプロバイダー（OpenAI、Anthropicなど） |
