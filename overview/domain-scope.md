# 도메인 스코프 설정

## Active Scope란?

**Active Scope**는 현재 화면에 표시되는 데이터의 범위를 결정합니다. 특정 도메인을 선택하면 모든 페이지의 데이터가 해당 도메인과 관련된 것만 표시됩니다.

## 사용법

### 도메인 선택

1. 화면 상단 헤더의 **ACTIVE SCOPE** 카드를 클릭
2. 드롭다운에서 원하는 도메인 선택
3. 모든 페이지(요약 보고, 파인딩, 다크웹 등)의 데이터가 즉시 필터링됨

### 스코프 해제

* **전체 보기**: 스코프 카드에서 "전체" 또는 X 버튼을 클릭
* 스코프가 해제되면 조직 전체의 데이터가 표시됩니다

## 영향 범위

Active Scope를 설정하면 다음 페이지들이 모두 영향을 받습니다:

| 페이지 | 영향 |
|--------|------|
| 요약 보고 | TOP 5, 통계, KPI 모두 해당 도메인 기준 |
| 파인딩 목록 | 해당 도메인의 발견사항만 표시 |
| 다크웹 | 해당 도메인의 다크웹 유출만 표시 |
| 모의침투 | 해당 도메인 자동 선택 |

## 팁

* 특정 도메인의 보안 상황을 집중적으로 확인할 때 사용하세요
* 보고서 작성 시 도메인별로 스코프를 변경하며 스크린샷을 캡처하면 효율적입니다
