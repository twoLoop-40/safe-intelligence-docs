# 자산 노출 현황 (Asset Exposure)

## 개요

요약 보고의 두 번째 섹션으로, 모니터링 중인 도메인들의 보안 상태를 카드 그리드로 보여줍니다.

## 도메인 카드

각 도메인은 하나의 카드로 표현됩니다:

* **보안 등급 배지**: A(초록) ~ F(빨간) 원형 배지
* **도메인 이름**: 고정폭 글꼴로 표시
* **서브도메인 수**: "N subdomains" 형태
* **심각도 배지**: Critical(빨간), High(주황) 발견사항 수
* **전체 발견사항 수**: 큰 숫자로 표시
* **마지막 스캔 시간**: 상대 시간 표시 (예: "5분 전")

## 카드 클릭

카드를 클릭하면 해당 도메인의 **도메인 상세** 페이지로 이동합니다. 해당 페이지에서 서브도메인, 발견사항, 스캔 이력 등을 자세히 확인할 수 있습니다.

## 전체 도메인 보기

**View All** 링크를 클릭하면 **도메인 목록** 페이지로 이동하여 모든 도메인을 확인할 수 있습니다.

## 보안 등급 기준

| 등급 | 점수 범위 | 색상 | 의미 |
|------|----------|------|------|
| A | 75+ | 초록 | 양호 |
| B | 60-74 | 연두 | 보통 |
| C | 40-59 | 노랑 | 주의 |
| D | 20-39 | 주황 | 위험 |
| F | 0-19 | 빨강 | 심각 |
