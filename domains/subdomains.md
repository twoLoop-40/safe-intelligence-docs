# 서브도메인 관리

## 개요

각 도메인 하위의 서브도메인을 관리하고 모니터링합니다. 도메인 상세 페이지의 **Subdomains** 탭에서 확인할 수 있습니다.

## 서브도메인 목록

각 서브도메인에 대해 다음 정보가 표시됩니다:

| 열 | 설명 |
|----|------|
| 이름 | 서브도메인 주소 (예: api.example.com) |
| IP | 확인된 IP 주소 |
| 상태 | 활성/비활성 |
| Critical | Critical 발견사항 수 |
| High | High 발견사항 수 |
| 포트 | 열린 포트 수 |
| SSL | SSL 인증서 상태 |
| 최근 스캔 | 마지막 스캔 시간 |

## 서브도메인 상세

서브도메인을 클릭하면 상세 페이지로 이동합니다:

* **기본 정보**: IP, 위치(GeoIP), 호스팅 정보
* **포트 정보**: 열린 포트와 실행 중인 서비스
* **발견사항**: 해당 서브도메인의 보안 이슈
* **SSL 인증서**: 인증서 세부 정보와 만료일

## 서브도메인 추가

도메인 상세 페이지에서 서브도메인을 수동으로 추가할 수 있습니다:

1. Subdomains 탭 이동
2. **+ 서브도메인 추가** 입력 필드에 서브도메인 이름 입력
3. 추가 버튼 클릭

{% hint style="info" %}
보안 스캔을 실행하면 서브도메인이 자동으로 발견되어 추가됩니다.
{% endhint %}
