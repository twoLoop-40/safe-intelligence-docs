# 도메인 추가

## 새 도메인 등록

모니터링할 도메인을 추가하는 방법입니다.

### 방법 1: 도메인 목록에서 추가

1. 사이드바에서 **Domains** 클릭
2. **+ 도메인 추가** 버튼 클릭
3. 도메인 이름 입력 (예: example.com)
4. **추가** 버튼 클릭

### 방법 2: 스캔으로 자동 추가

새 스캔을 실행할 때 도메인을 입력하면 자동으로 등록됩니다.

1. 사이드바에서 **New Scan** 클릭
2. 대상 도메인 입력
3. 스캔 실행 → 도메인과 서브도메인이 자동 등록

## 주의사항

* 본인 조직이 소유한 도메인만 등록하세요
* 등록된 도메인은 모든 팀원에게 공유됩니다
* 도메인당 서브도메인은 자동 탐색됩니다 (Subfinder 기반)
