# 도메인 목록

## 개요

모니터링 중인 모든 도메인을 한 화면에서 관리합니다. 사이드바의 **Domains** 메뉴에서 접근할 수 있습니다.

## 도메인 카드

각 도메인은 카드 형태로 표시됩니다:

* **보안 등급** (A~F): 색상 코딩된 배지
* **도메인 이름**: 고정폭 글꼴
* **서브도메인 수**: 하위 서브도메인 개수
* **Critical/High 발견사항**: 심각도별 배지
* **전체 발견사항 수**: 큰 숫자
* **마지막 스캔 시간**: 상대 시간

## 도메인 상세 보기

카드를 클릭하면 **도메인 상세** 페이지로 이동합니다.

## 도메인 추가

**+ 도메인 추가** 버튼을 클릭하여 새 도메인을 등록할 수 있습니다. 자세한 내용은 [도메인 추가](add-domain.md) 페이지를 참조하세요.

## 정렬 및 필터

* 보안 등급, 발견사항 수, 도메인 이름 등으로 정렬 가능
* 검색 바를 사용하여 특정 도메인을 빠르게 찾을 수 있습니다
