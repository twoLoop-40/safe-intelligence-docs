# 도메인 상세

## 개요

특정 도메인의 보안 현황을 자세히 확인하는 페이지입니다. 도메인 목록이나 요약 보고에서 도메인을 클릭하면 이동합니다.

## 화면 구성

### 상단: 도메인 정보

* **도메인 이름**: 현재 조회 중인 도메인
* **보안 점수**: 0~~100 숫자와 등급 (A~~F)
* **통계 카드**: 서브도메인 수, 발견사항 수, 최근 스캔 시간

### 탭 구성

| 탭          | 내용                       |
| ---------- | ------------------------ |
| Overview   | 도메인 보안 요약, 심각도 분포, 최근 활동 |
| Findings   | 이 도메인의 모든 발견사항 목록        |
| Dark Web   | 이 도메인 관련 다크웹 유출 현황       |
| Subdomains | 하위 서브도메인 목록 및 상태         |

### Attack Surface 섹션

도메인의 공격 표면을 시각적으로 보여줍니다:

* **열린 포트**: 탐지된 네트워크 서비스 목록
* **SSL 인증서**: 인증서 상태 및 만료일
* **웹 기술**: 감지된 웹 서버, 프레임워크 등

## 서브도메인 탐색

서브도메인 탭에서 개별 서브도메인을 클릭하면 서브도메인 상세 페이지로 이동합니다. 각 서브도메인의:

* IP 주소
* 서비스 상태
* 보안 발견사항
* 포트 정보

를 확인할 수 있습니다.

## 스캔 실행

도메인 상세 페이지에서 **스캔 시작** 버튼을 클릭하여 해당 도메인에 대한 보안 스캔을 바로 실행할 수 있습니다.
