# 문제 해결

## 로그인 문제

### 로그인이 되지 않습니다

* 이메일 주소와 비밀번호를 확인하세요
* Caps Lock이 켜져 있지 않은지 확인하세요
* 비밀번호를 분실한 경우 관리자에게 문의하세요

### 세션이 자주 만료됩니다

보안을 위해 일정 시간 비활성 상태가 지속되면 자동으로 로그아웃됩니다. 다시 로그인하여 계속 사용하세요.

## 데이터 표시 문제

### 데이터가 표시되지 않습니다

* 도메인이 올바르게 등록되어 있는지 확인하세요
* 최초 스캔이 완료될 때까지 기다려 주세요 (최대 10분)
* 브라우저를 새로고침해 보세요

### 스코어가 업데이트되지 않습니다

스코어는 스캔 완료 후 자동으로 재계산됩니다. 스캔이 진행 중인 경우 완료를 기다려 주세요.

### 서브도메인이 누락되었습니다

자동 발견으로 찾지 못한 서브도메인은 수동으로 추가할 수 있습니다. 도메인 상세 페이지에서 서브도메인을 직접 입력하세요.

## 스캔 문제

### 스캔이 시작되지 않습니다

* 이미 진행 중인 스캔이 있는지 확인하세요
* 네트워크 연결 상태를 확인하세요

### 스캔이 오래 걸립니다

서브도메인이 많은 도메인의 경우 스캔에 시간이 더 소요될 수 있습니다. 일반적으로 10분 이내에 완료됩니다.

## 브라우저 호환성

### 지원 브라우저

* Chrome (최신 버전 권장)
* Firefox
* Safari
* Edge

### 권장 사항

* JavaScript가 활성화되어 있어야 합니다
* 쿠키를 허용해 주세요
* 팝업 차단기가 SafeIntelligence 도메인을 허용하는지 확인하세요

## 추가 지원

위 방법으로 해결되지 않는 문제가 있다면 관리자에게 문의하세요.
