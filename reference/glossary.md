# 용어 정의

## 일반 용어

| 용어 | 설명 |
|------|------|
| **ASM** | Attack Surface Management. 공격 표면 관리. 외부에 노출된 자산을 식별하고 보안 위험을 관리하는 활동 |
| **OSINT** | Open Source Intelligence. 공개 출처 정보를 활용한 인텔리전스 수집 |
| **CTI** | Cyber Threat Intelligence. 사이버 위협 인텔리전스 |

## 자산 관련

| 용어 | 설명 |
|------|------|
| **도메인** | 조직이 소유한 인터넷 도메인 (예: example.com) |
| **서브도메인** | 도메인 하위의 호스트명 (예: api.example.com) |
| **스코프** | 모니터링 대상으로 설정된 도메인 범위 |
| **자산(Asset)** | 도메인, 서브도메인, IP 등 모니터링 대상 리소스 |

## 취약점 관련

| 용어 | 설명 |
|------|------|
| **CVE** | Common Vulnerabilities and Exposures. 공개 알려진 보안 취약점 식별자 |
| **CVSS** | Common Vulnerability Scoring System. 취약점 심각도 점수 (0-10) |
| **CWE** | Common Weakness Enumeration. 소프트웨어 보안 약점 분류 체계 |
| **CPE** | Common Platform Enumeration. 소프트웨어/하드웨어 식별 표준 |

## 심각도

| 등급 | CVSS 범위 | 설명 |
|------|----------|------|
| **Critical** | 9.0 - 10.0 | 즉시 대응 필요한 치명적 취약점 |
| **High** | 7.0 - 8.9 | 빠른 대응이 필요한 높은 위험 |
| **Medium** | 4.0 - 6.9 | 계획적 대응이 필요한 중간 위험 |
| **Low** | 0.1 - 3.9 | 낮은 위험, 모니터링 수준 |

## 다크웹 관련

| 용어 | 설명 |
|------|------|
| **다크웹** | 일반 브라우저로 접근할 수 없는 인터넷 영역 |
| **자격증명 유출** | 이메일+비밀번호가 다크웹에서 발견된 경우 |
| **Credential Stuffing** | 유출된 자격증명을 다른 서비스에 대입하는 공격 |
| **감염 디바이스** | 멀웨어에 감염되어 정보가 유출된 기기 |
| **피싱 키트** | 피싱 공격에 사용되는 도구 모음 |
