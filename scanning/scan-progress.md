# 스캔 진행 상황

## 진행률 표시

스캔이 시작되면 헤더 영역에 진행률이 표시됩니다:

* **진행률 바**: 전체 스캔 진행 비율
* **현재 단계**: 현재 수행 중인 스캔 유형 (예: "서브도메인 발견 중...")
* **경과 시간**: 스캔 시작 후 경과 시간

## 스캔 단계

스캔은 아래 순서로 진행됩니다:

1. **초기화** — 대상 도메인 및 서브도메인 목록 준비
2. **서브도메인 발견** — 새로운 서브도메인 탐색
3. **포트 스캔** — 각 서브도메인의 오픈 포트 확인
4. **서비스 탐지** — 오픈 포트의 서비스 식별
5. **취약점 매칭** — CVE 데이터베이스와 매칭
6. **인증서 검증** — SSL/TLS 상태 확인
7. **결과 집계** — 스코어 계산 및 발견사항 생성

## 스캔 완료

스캔이 완료되면:

* 진행률 바가 100%로 표시됩니다
* 새로 발견된 사항의 개수가 안내됩니다
* 도메인 스코어가 자동으로 재계산됩니다
* 요약 보고(Overview)의 데이터가 갱신됩니다

## 스캔 실패 시

네트워크 오류 등으로 스캔이 실패한 경우:

* 오류 메시지가 표시됩니다
* 부분적으로 수집된 데이터는 유지됩니다
* 잠시 후 다시 시도할 수 있습니다
