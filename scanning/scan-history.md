# 스캔 이력

## 개요

과거 실행된 모든 스캔의 이력을 확인할 수 있습니다. 최대 6개월간의 스캔 기록이 보관됩니다.

## 이력 정보

각 스캔 기록에는 다음 정보가 포함됩니다:

| 항목 | 설명 |
|------|------|
| 시작 시간 | 스캔이 시작된 일시 |
| 완료 시간 | 스캔이 종료된 일시 |
| 대상 도메인 | 스캔 대상 도메인 |
| 상태 | 완료 / 실패 / 취소 |
| 발견 항목 수 | 새로 발견된 보안 사항 수 |

## 자동 스캔

SafeIntelligence는 등록된 도메인에 대해 자동 스캔을 수행합니다:

* **보안 스캔**: 주기적 자동 실행
* **다크웹 모니터링**: 6시간 주기 자동 갱신
* **인증서 만료 확인**: 매일 확인

## 보관 기간

* 스캔 결과: **6개월** 보관
* 발견사항(Findings): 해결될 때까지 영구 보관
* 스코어 이력: 영구 보관
